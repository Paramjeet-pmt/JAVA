# JAVA Programs

A collection of beginner-friendly Java programs I’ve written while learning the language.

## Programs Included
- **helloworld.java** → Prints "Hello, World"
- **basic.java** → Basic Java syntax practice
- **evennotilln.java** → Prints even numbers up to N
- **lesson.java / lesson3.java** → Practice exercises
- **monthprint.java** → Prints months (with logic examples)
